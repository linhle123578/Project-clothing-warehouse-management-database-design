Cách để chạy file

B1: Mở SQL Server Management Studio và chọn New Query

B2: Mở file "SCRIPT DỮ LIỆU MẪU" và copy phần script vào ứng dựng SQL Server Management Studio 

B3: Chạy các câu lệnh để tạo database và tạo bảng dữ liệu

B4: Mở file "SCRIPT TẠO BẢNG" và copy phần script vào ứng dựng SQL Server Management Studio 

B5: Chạy các câu lệnh để cũng cấp các dữ liệu thống kê
